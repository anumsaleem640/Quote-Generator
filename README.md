# Quote Generator

A full-stack quote application built with the MERN stack and React Native.
Delivers personalized quotes to users based on their selected categories,
with a free/premium subscription model.

## Architecture

- **Frontend** — React + Vite + Bootstrap (marketing site + user dashboard)
- **Backend** — Node.js + Express.js + MongoDB + JWT authentication
- **Mobile** — React Native CLI (iOS and Android)

## Repository Structure

| Folder | Purpose |
|--------|---------|
| `frontend/` | React web application |
| `backend/` | Express.js REST API |
| `mobile-app/` | React Native mobile app |
| `docs/` | Architecture, API, and deployment docs |

## Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn
- MongoDB Atlas account (free)
- VS Code

### Setup
1. Clone this repository
2. Copy `.env.example` to `.env` in each sub-folder and fill in your values
3. Run `npm install` in each sub-folder
4. See each folder's README for specific run instructions
